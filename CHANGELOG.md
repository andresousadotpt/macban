## [0.1.0] - 2026-06-11

### Added

- Local-first kanban with per-column JSON storage and sync-friendly file layout.
- Native SwiftUI board UI with drag-and-drop, card detail editor, and project settings.
- `MacbanCore` + `Macban` SwiftPM targets with CI and Homebrew release pipeline.
